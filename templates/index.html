  • /api/login_user/?username=FooFighter&email=foo@gmail.com&password=Password1234567890!&remember=True

  • /api/update_user/?userId=3&username=OptionalUsername&email=OptionalEmail&bio=OptionalBio&password=Password1234567890!
  • /api/load_userId/?username=user_1

  • /api/load_user_groups/?userId=2

    • Load Feed

    • Time: /api/load_feed/?type=Time&userId=2
    • Location (Distance is optional): /api/load_feed/?type=Location&userId=2&location=Melkbos&distance=100
    • Category: /api/load_feed/?type=Category&userId=2&category=Lego
    • Group: /api/load_feed/?type=Group&userId=2&groupId=30
    • User: /api/load_feed/?type=User&userId=2&username=swinger_1
Group API

Group CRUD Operations

  • Create - POST: /api/group/?title=My_Title&description=My_description&private=True&userId=2
  • Read - GET: /api/group/?groupId=28
  • Update - PUT: /api/group/?groupId=28&title=My_Title&description=My_description&private=True
  • Delete - DELETE: /api/group/?groupId=25

Other Group Operations

  • /api/leave_group/?userId=2&groupId=29
  • /api/join_group/?userId=2&groupId=29

Post API

Post CRUD Operations

  • Create - POST: /api/post/?userId=2&groupId=18&title=Bush+Babies&postContent=I+love+bush+babies&location=Melkbos&category=Other
  • Read - GET: /api/post/?postId=18&userId=2
  • Update - PUT: /api/post/?postId=19&userId=2&title=My_Title&postContent=My_description
  • Delete - DELETE: /api/post/?postId=19&userId=2

Other Post Operations

  • /api/like_post/?userId=2&postId=13
  • /api/unlike_post/?userId=2&postId=13
Comment API
  • Create - POST: /api/comment/?postId=1&userId=1&content=You+look+terrifying
  • Read - GET: /api/comment/?commentId=1
  • Update - PUT: /api/comment/?commentId=1&content=You+look+so+cute
  • Delete - DELETE: /api/comment/?commentId=1
Note API
  • Create - POST: /api/note/?userId=1&groupId=14&type=0&desc=You+have+a+friend+invite
  • Read - GET: /api/note/?noteId=1
  • Update - PUT: /api/note/?noteId=1
  • Delete - DELETE: /api/note/?noteId=1
