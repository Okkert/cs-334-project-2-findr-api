Group API

Group CRUD Operations

  • Create - POST: /api/group/?title=My_Title&description=My_description&private=True
  • Read - GET: /api/group/?groupId=13
  • Update - PUT: /api/group/?groupId=13&title=My_Title&description=My_description&private=True
  • Delete - DELETE: /api/group/?groupId=12

Other Group Operations

  • /api/leave_group/?userId=2&groupId=11
  • /api/join_group/?userId=2&groupId=13
  • /api/load_group_posts/?groupId=13
  • /api/load_group_members/?groupId=13
  • /api/search_groups/?title=hot+tub
  • /api/promote_member/?groupId=13&userId=2&promoteId=3
  • /api/demote_member/?groupId=13&userId=2&demoteId=3
  • /api/load_join_request/?groupId=13
  • /api/request_join_group/?groupId=13&userId=2
  • /api/accept_join_request/?groupId=13&userId=2
  • /api/decline_join_request/?groupId=13&userId=2
Post API

Post CRUD Operations

  • Create - POST: /api/post/?userId=2&groupId=18&title=Bush+Babies&description=I+love+bush+babies&location=Melkbos&category=Other
  • Read - GET: /api/post/?groupId=13
  • Update - PUT: /api/post/?groupId=13&title=My_Title&description=My_description&private=True
  • Delete - DELETE: /api/post/?groupId=12

Other Group Operations

  • /api/like_post/?userId=2&postId=13

Comment API

  • Create - POST: /api/comment/?postId=1&userId=1&content=You+look+terrifying
  • Read - GET: /api/comment/?commentId=1
  • Update - PUT: /api/comment/?commentId=1&content=You+look+so+cute
  • Delete - DELETE: /api/comment/?commentId=1

Note API

  • Read - GET: /api/note/?noteId=1
  • Update - PUT: /api/note/?noteId=1
  • Delete - DELETE: /api/note/?noteId=1
