Welcome to the CS334 Project 3 Findr API Routes

API Routes:

User API

  • /api/register_user/?username=FooFighter&email=foo@gmail.com&password=Password1234567890!
  • /api/login_user/?username=FooFighter&email=foo@gmail.com&password=Password1234567890!&remember=True
  • /api/load_user/?userId=1
  • /api/update_user/?userId=3&username=OptionalUsername&email=OptionalEmail&bio=OptionalBio&password=Password1234567890!

Group API

  • /api/create_group/?title=My_Title&description=My_description&private=True
  • /api/edit_group/?groupId=13&title=My_Title&description=My_description&private=True
  • /api/leave_group/?userId=2&groupId=11
  • /api/delete_group/?groupId=12
  • /api/join_group/?userId=2&groupId=13
  • /api/load_group/?groupId=13
  • /api/load_group_posts/?groupId=13
  • /api/load_group_members/?groupId=13
  • /api/search_groups/?title=hot+tub
  • /api/promote_member/?groupId=13&userId=2&promoteId=3
  • /api/demote_member/?groupId=13&userId=2&demoteId=3

Post API

  • /api/create_post/?userId=2&groupId=13&title=Fun+fact+of+the+day&postContent=I+like+hot+tubs&location=Stellenbosch&category=Other

Comment API

  • Create - POST: /api/comment/?postId=1&userId=1&content=You+look+terrifying
  • Read - GET: /api/comment/?commentId=1
  • Update - PUT: /api/comment/?commentId=1&content=You+look+so+cute
  • Delete - DELETE: /api/comment/?commentId=1
